# React, Node Template


A React, Node with these technologies:

#### For Front-End
- React
- TypeScript
- Sass
- Vite

#### For Back-End
- Node
- TypeScript
- Nodemon
- ts-node
- Express

#### Formatters & Linters
- Prettier
- ESLint
- Stylelint
- CSpell
